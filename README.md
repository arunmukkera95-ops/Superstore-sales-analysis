1. Project Title
#Superstore Sales Analysis Dashboard

2. Project Overview
##This Projects analyzes Superstore sales data to uncover insights into revenue trends, product performance, and customer behavior using python, MySQL, and Power BI.

3. Objectives
- Analyzes sales trends over time
- Identify top-performing products
- Understand customer demographics
- Evaluate profitability and business performance

4. Tools & Technologies
- Python (Pandas, NumPy)
- MySQL
- Power BI
- Excel / CSV

5. Data Pipeline
1. Raw CSV data collected
2. Cleaned Data using python
3. Data stored in MySQL database
4. SQL queries used for analysis
5. Dashboard built in Power BI

6. Data Cleaning Steps
- standardized all column names 
- standardized date columns 
- converted date column to US format first 
- Handled missing values using datetime function

7. Key Analysis (SQL + Python)
- Monthly Sales Trend analysis
- Yearly Sales Trend analysis
- Top and Bottom Products analysis
- sales by category, city, region, 
- customer segmentation (customers)
- profitability analysis

8. Key Insights
- Sales show a consistent upward trend over time, indicating steady business growth.
- Top performing products contribute a significant share of total revenue, highlighting - product concentration.
- The Technology category drives the highest sales, outperforming other categories.
- The West region is the leading contributor to overall sales.
- Certain cities outperform others, indicating strong localized demand.
- There are growth opportunities in low-performing products through targeted strategies.
- The top 10 customers account for a substantial portion of total sales, indicating customer concentration.
- The Consumer segment contributes approximately 50% of total revenue, making it the dominant segment.
- New York and California together contribute around 35% of total sales, making them key markets.
- Phones and Chairs together account for approximately 29% of total sales, making them top-performing product categories.

9. How to Run
1. Clone the repository
2. Run Python script for data cleaning
3. Load data into MySQL
4. Open Power BI Dashboard
